### 说明

技术栈：react全家桶+antd+axios+ts

添加npm包管理

1、安装依赖

```bash
yarn
```

2、启动项目

```bash
yarn start
```

3、构建项目

```bash
yarn build
```

4、项目目录结构

```bash
.
├── README.md
├── dist
├── public
├── src
│   ├── assets
│   ├── components
│   ├── app.tsx
│   ├── layout.tsx
│   ├── pages
│   ├── routers.tsx
│   └── utils
└── tsconfig.json

```
